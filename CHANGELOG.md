
# 版本更新日志 (CHANGELOG)
##本次核心升级 (v0.8 → v0.8.1) 四大重磅特性

-✅ 自动 TTS 服务存活检测：程序启动自动检测 127.0.0.1:9666 TTS 服务是否在线，服务离线则自动静默关闭 TTS 功能，无报错、无缝降级，服务恢复重启程序即可启用

-✅ 命令行参数化开关：所有核心功能由「命令行传参动态控制」，彻底替代硬编码宏定义，无需修改源码 / 重新编译，一行命令控制所有开关，灵活度拉满

-✅ 可选强制关闭 TTS：支持命令行手动关闭 TTS，优先级高于自动检测，满足无 TTS 需求的场景

-✅ 完全兼容 v08 全部功能：保留原生中文无乱码 / RC4 加密 / 托盘 / 全局热键 / 组播 / 自动重连 / 内存≤5M 等所有特性，零侵入式升级、无代码冗余、无性能损耗

-✅ 新增特性：静默启动、音效开关、强制 UTF8、帮助指令等实用命令行参数，生产级完善度
